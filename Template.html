変数

テンプレートファイルでは以下の変数が使えます[1]

Name Syntax
{{{ contents }}} マークダウンファイルを変換したHTMLが入ります
{{ title }} ファイル名
{{ date }} 変換日時
{{{ mathcss }}}

数式用のスタイルシートを埋め込みます。

この変数が有効なのは --math オプションが未指定で、かつ元の文書にブロック数式が使われている場合のみです。それ以外の条件では空文字になります。

入力にディレクトリを指定した場合、内部では2パス描画が有効になります。
一度全てのファイルを空変換し、必要最小限で完全なスタイルシートを取得します。

--watchモードでは変更のあったファイルだけが再生成され、スタイルシートは不要になった文字のスタイルも含まれるため、最終描画結果にばらつきが生じます。

ファイル参照

<a href="@/css/style.css">

@/ から始まるパスは、入力ファイルの位置によらず、出力ディレクトリを基準としたファイルを参照します。

パスの検索には正規表現を使っており、場合によっては置換されない可能性があります。本ツールは次の属性の値をパスとみなします。

href
src
action
formaction
poster
cite
data
manifest
srcset
imagesrcset
ping
content
usemap
