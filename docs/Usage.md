
# 使い方

```sh
mdhtml <input> [options]
```

```table width=100% cols="25,75"
| `<input>`
| 入力ファイル/フォルダを指定

入力がファイルの場合、そのファイルを変換します。  
入力がフォルダの場合、そのフォルダ内の全ての文書を変換します。
```


## 設定一覧

```table cols="25,5,70"

|`--output`
| `-o`
| 出力ファイル/フォルダを指定

入力がファイルの場合、出力ファイル名を指定してください。  
入力がフォルダの場合、出力先のフォルダ名を指定してください。

|`--template`
|`-t`
| テンプレートファイルを指定

| `--config`
| `-c`
| 設定ファイル(JSON)を読み込みます

設定ファイルの内容はコマンドラインの指定で上書きされます。

| `--ignore`
| `-i`
| 変換対象から除外するパターン(path,glob)を指定します

複数回指定できます。

| `--clean`
| 
| 変換前に出力先のフォルダを全消去します

|`--quiet`
|`-q`
| 変換メッセージをOFFにします

| `--watch`
| `-w`
| ウォッチモードを有効にします

ウォッチモードでは入力ファイルを監視し、ファイル変更を検知すると自動的にそのファイルを変換します。

| `--math [path]`
| 
| 数式用のCSSファイルを出力します (パス省略時: math.css)

本文に数式を使う場合、数式描画用のCSSファイルが必要です。

本ツールはインライン数式に katex を、ブロック数式に mathjax を使います。これらを表示するフォントとスタイルシートは CDN からダウンロードする想定ですが、mathjax のスタイルシートはサイズが大きく、オンラインで配信されていません。本オプションを利用すると入力ファイルを表示できる必要最小限のスタイルシートを生成します。

```

## 高度な設定

```table cols="25,65"
| `--math-font-url`
| MathJax のフォントURLを変更します。

初期値:  `https://cdn.jsdelivr.net/npm/@mathjax/mathjax-newcm-font@latest/`
```
